# サービス一覧

## 使用中のサービス

| サービス | 何をしたか | URL |
|----------|-----------|-----|
| **Render** | アプリ本体のホスティング・デプロイ | https://render.com |
| **Cloudflare Pages** | ランディングページ（`landing/index.html`）のホスティング | https://pages.cloudflare.com |
| **Stripe** | 月額課金（Proプラン）・Webhook設定・価格ID発行 | https://dashboard.stripe.com |
| **Gmail** | パスワードリセットメール送信用SMTPアカウント設定 | https://myaccount.google.com/apppasswords |
| **Google Analytics** | アクセス解析タグ設置（ID: `G-1MHRVYZ2WV`） | https://analytics.google.com |

## 各サービスで確認すべき設定

| サービス | 確認ポイント |
|----------|------------|
| Render | 環境変数（`SECRET_KEY`, `STRIPE_*`, `MAIL_*`, `ADMIN_PASSWORD`）が設定されているか |
| Stripe | Webhookエンドポイント `https://taskmanagement-1rss.onrender.com/webhook` が登録されているか |
| Gmail | アプリパスワードが発行済みか（通常パスワードではNG） |
| Cloudflare | `landing/index.html` の最新版がデプロイされているか |

## 環境変数一覧（Renderに設定するもの）

| 変数名 | サービス | 用途 |
|--------|---------|------|
| `SECRET_KEY` | Flask | セッション暗号化 |
| `STRIPE_SECRET_KEY` | Stripe | 決済処理 |
| `STRIPE_PUBLISHABLE_KEY` | Stripe | フロントエンド決済 |
| `STRIPE_WEBHOOK_SECRET` | Stripe | Webhook署名検証 |
| `STRIPE_PRICE_ID` | Stripe | Proプラン価格ID |
| `MAIL_USERNAME` | Gmail | 送信元メールアドレス |
| `MAIL_PASSWORD` | Gmail | アプリパスワード |
| `ADMIN_PASSWORD` | Flask | 管理画面ログインパスワード |
